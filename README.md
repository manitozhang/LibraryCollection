## 项目介绍
+ 针对用于一些外包或者其他快速开发的app,项目中包含了一些常用的框架,还有集成一些常用的第三方库以及使用,以及常用的组合控件(标题栏,按钮,菜单条目...),每个第三方使用方法和项目的架构均可直接复制到项目中使用,没有很多关联,继承实现之类,基本上每个东西单独封装,可直接引入basecommon包使用,也可按需cv
